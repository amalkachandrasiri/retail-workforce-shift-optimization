# retail-workforce-shift-optimization
Optimizing employee shift assignments in a retail environment.

# Project Overview
This project focuses on optimizing employee shift assignments in a retail environment using two optimization techniques:
Genetic Algorithm (GA)
Mixed Integer Programming (MIP)
The goal is to generate cost-effective workforce schedules while satisfying staffing requirements and employee availability constraints.

# Problem Statement
Retail stores must assign employees to shifts while ensuring sufficient staffing levels and minimizing labor costs.

# Objective
Minimize total labor cost while meeting operational staffing requirements.
